# CSS Landing Projects

A collection of landing pages built while learning 
**CSS Responsive Design** with a **mobile-first approach**.

## What is this?
These are projects I built while practicing and revising 
CSS concepts — focusing on making layouts that look great 
on all screen sizes, starting from mobile.

## Concepts Practiced
- Mobile-first design
- Flexbox & Grid
- Media queries
- Responsive typography
- CSS variables
- Clean UI layouts

## Goal
Each project helped me understand how to build real-world 
responsive pages from scratch using pure HTML & CSS.
